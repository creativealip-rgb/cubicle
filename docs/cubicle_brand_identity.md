# Cubicle Brand Identity

## Positioning

Cubicle is a client operations hub for freelancers, agencies, consultants, and small service teams.

One-liner:
> Manage client work from request to invoice.

Short description:
> Cubicle brings clients, projects, tasks, files, time tracking, invoices, booking, and client portals into one calm workspace built for client-service businesses.

## Category

Primary category:
- Client Operations Hub

Secondary phrases:
- Client Work OS
- Client work management
- Project delivery workspace
- Client portal + invoicing workspace

Avoid primary category:
- Generic project management
- Task management
- ClickUp alternative
- Asana alternative

## Strategic Angle

ClickUp and Asana are built for internal team project management.
Cubicle is built for client-service work where projects end with deliverables, client updates, tracked time, and invoices.

Core contrast:
> Not another task board. A workspace for running client work from request to invoice.

## Target Audience

Best-fit customers:
- Freelancers
- Creative agencies
- Web/design studios
- Software studios
- Marketing teams
- Consultants
- Small client-service teams

Not primary target:
- Enterprise PM teams
- Internal engineering teams
- Large product orgs
- Generic task-only users

## Brand Promise

Cubicle helps small service teams stop juggling client work across chat, spreadsheets, file drives, invoice tools, and project boards.

## Messaging Pillars

1. Client-centered workspace
   - Every project starts from a client, not a random board.

2. Work delivery, not task chaos
   - Tasks, files, comments, time, and invoices stay connected.

3. Client portal by default
   - Clients get one clean place to see what matters.

4. Billing-ready operations
   - Time tracking and invoices sit beside work, not after it.

5. Calm professional UI
   - Built for clarity, handover, and trust.

## Hero Copy

Recommended subheadline:
